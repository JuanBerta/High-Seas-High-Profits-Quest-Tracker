# High Seas, High Profits - Quest Tracker

A productivity-focused mod for **High Seas, High Profits** developed with MelonLoader and Harmony. This mod adds a persistent, organized tracker to your HUD to help you manage Mayor Quests across multiple cities without needing to constantly check the Town Hall.

## 🚀 Features

*   **Multi-City Tracking:** Track multiple quests simultaneously across different ports.
*   **City Grouping:** Quests are automatically grouped by city name in the HUD for better logistics planning.
*   **Auto-Cleanup:** Quests are automatically removed from your tracker the moment they are completed.
*   **Native UI Integration:** Adds a "Track/Untrack" toggle directly into the Town Hall quest list.

## 🛠️ Installation

1. Install [MelonLoader](https://melonwiki.xyz/).
2. Place the compiled `Goods Tracker.dll` into your game's `Mods` folder.
3. Launch the game and visit any Town Hall to start tracking!

## 💻 Tech Stack

*   **Language:** C#
*   **Framework:** MelonLoader
*   **Patching:** Harmony (Lib)
*   **UI:** Unity TextMeshPro
